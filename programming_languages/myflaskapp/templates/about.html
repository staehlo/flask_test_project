This web application is a test project. The focus lies on the technical implementation of its features and not on the design.

The following features were implemented:

  • Combining Python's Flask web framework with the DataTables plug-in for the jQuery Javascript library
  • Creating plots with Python's matplotlib library
  • Combining Flask with Python's plotly library for interactive chart presentation
  • Deploying a Flask application on pythonanywhere

The steps to produce this application were:

  • Simple copy & paste of all tables from Wikipedia's Timeline of programming languages
    (date retrieved: 24 June 2022)
  • Cleaning up the data and solving naming inconsistencies in Excel
  • Importing the data into an SQL database
  • Creating an Object relational mapping with sqlacodegen
  • Writing the Flask app
  • Deploying the Flask app
The code of this application can be found on GitHub:
GitHub.com/staehlo/flask_test_project
